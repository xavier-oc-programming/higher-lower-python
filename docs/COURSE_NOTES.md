# Day 14 — Higher or Lower

## Course Exercise

Build a terminal game that asks the user to guess which of two Instagram accounts
has more followers.

### Rules
- Two accounts are shown side by side with a `vs` separator.
- The user types `A` or `B`.
- Correct guess → score goes up, account B becomes the new A, a fresh B is drawn.
- Wrong guess → game over, final score is displayed.

### Reference
- Original game: https://www.higherlowergame.com/
- Course demo: https://appbrewery.github.io/python-day14-demo/

## Concepts Practiced
- Functions and return values
- Importing modules (`art`, `game_data`)
- `while` loops and game loops
- `random.choice()`
- String formatting with f-strings
- Input validation
